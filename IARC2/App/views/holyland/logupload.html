Holyland Contest - Log Upload

Log files will be accepted until May 31st.



Holyland Cabrillo Format


START-OF-LOG: version-number X.X

Must be the first line of the log submission.
Must be version 3.0

CALLSIGN: callsign

The callsign used during the contest.

CONTEST: contest-name

The contest-name must be HOLYLAND

CATEGORY-OPERATOR: text

The category-operator must be one of the following:
SINGLE-OP
MULTI-OP
SWL
CHECKLOG

CATEGORY-MODE: text

The category-mode must be one of the following:
SSB
CW
DIGI
MIXED

CATEGORY-POWER: text

The category-power must be one of the following:
HIGH
QRP

CREATED-BY: text 

Name and version of the logging program used to create the Cabrillo file. (This field is optional)

EMAIL: text

A place to put an email address where we can contact you if there is a question about your log. (This field is optional)

NAME: text 

Your name.

OPERATORS: callsign1, [callsign2, callsign3...]

A space-delimited list of operator callsigns for multi operator stations
NOTE: If you select a multi-operator category and do not enter operator callsigns in this field, the robot will reject your log. Please use a comma between the operator calls.

QSO: qso-data

The qso-data format is shown below.
  -----Data Sent----- -----Data Rcvd-----
QSO: freq mo date time my call rst exch dx call rst exch
QSO: 14240 PH 2000-11-26 0711 N6TW 59 03 4Z5SL 59 K01YZ

END-OF-LOG:

Must be the last line of the log submission.
